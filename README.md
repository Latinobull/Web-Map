> This is an application where you can see all volcanoes in the United States. As well, each country is colored based on its population size. The webpage was made and powered by the folium library.

### 🏠 [Homepage](https://usa-volcanoes.herokuapp.com/)

## Prerequisites

- python >=3.0.0

## Install

```sh
pip install
```

## Usage

```sh
py script.py
```
